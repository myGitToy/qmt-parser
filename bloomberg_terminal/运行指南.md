# 🚀 Bloomberg Terminal Demo - 运行指南

## ✅ 解决方案

您遇到的错误是因为尝试用Python运行了`.bat`文件。现在有多种启动方式可选：

## 📋 启动方式

### 方式1: 在VS Code中直接运行 (推荐)

1. 在VS Code中打开 `demo_simple.py` 文件
2. 点击右上角的 "▶️ 运行Python文件" 按钮
3. 或按 `Ctrl + F5` 快捷键

### 方式2: 使用终端命令

```bash
# 直接运行Streamlit
streamlit run demo_simple.py

# 或指定端口
streamlit run demo_simple.py --server.port 8501
```

### 方式3: 双击批处理文件

双击 `start_demo.bat` 文件（Windows系统）

### 方式4: 使用Python启动脚本

```bash
python start_demo.py
```

## 🔧 如果遇到依赖问题

### 安装Streamlit
```bash
pip install streamlit
```

### 安装所有依赖
```bash
pip install streamlit plotly pandas numpy
```

### 使用conda安装
```bash
conda install streamlit plotly pandas numpy
```

## 📱 访问应用

启动成功后，在浏览器中访问:
```
http://localhost:8501
```


**A**: 更换端口启动

**A**: 手动访问 http://localhost:8501
