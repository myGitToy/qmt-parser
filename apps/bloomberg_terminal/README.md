# Bloomberg Terminal Demo

一个模仿彭博终端的金融数据平台演示应用。

## 功能特点

🏦 **专业界面**: 模仿彭博终端的黑色主题界面  
📊 **实时数据**: 集成TuShare Pro和AKShare数据源  
📈 **K线图表**: 专业的金融图表展示  
🔬 **技术分析**: 移动平均线、RSI等技术指标  
💰 **关键指标**: 实时价格、涨跌幅、成交量等  

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行应用

```bash
streamlit run app.py
```

### 访问应用

打开浏览器访问: http://localhost:8501

## 技术架构

### 前端
- **Streamlit**: 快速构建Web应用
- **Plotly**: 交互式金融图表
- **CSS**: 自定义彭博终端风格样式

### 数据源
- **TuShare Pro**: 专业金融数据
- **AKShare**: 免费金融数据
- **模拟数据**: 数据源不可用时的备选方案

### 功能模块
- 股票选择和时间范围控制
- K线图和成交量图
- 技术指标分析(MA5, MA20, RSI)
- 实时价格和关键指标展示
- 历史数据表格

## 技术选型说明

### 为什么选择Streamlit?
1. **快速开发**: 纯Python开发，无需前端知识
2. **丰富组件**: 内置金融应用常用组件
3. **易于部署**: 一键部署到云平台
4. **良好生态**: 与pandas、plotly完美集成

### 替代方案
1. **Dash**: 更灵活但复杂度较高
2. **React + FastAPI**: 最专业但开发成本高
3. **Jupyter + Voila**: 适合研究但不适合生产

## 使用说明

### 股票代码格式
- 上海证券交易所: `600519.sh` (贵州茅台)
- 深圳证券交易所: `000858.sz` (五粮液)

### 时间范围
- 1个月: 近30个交易日
- 3个月: 近90个交易日  
- 6个月: 近180个交易日
- 1年: 近365个交易日

### 技术指标说明
- **MA5**: 5日移动平均线
- **MA20**: 20日移动平均线
- **RSI**: 相对强弱指数 (14日)
  - > 70: 超买区域
  - < 30: 超卖区域
